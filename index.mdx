## Why McDis-RCON?

<AccordionGroup>
  <Accordion title="Remote Server Management">
    Control your Minecraft servers from anywhere using Discord. No need for SSH or VNC connections - everything runs through a familiar Discord interface.
  </Accordion>
  <Accordion title="Multiple Server Support">
    Manage multiple Minecraft servers (Fabric, Paper, Vanilla, Velocity) simultaneously from a single Discord bot.
  </Accordion>
  <Accordion title="Advanced Error Reporting">
    Automatic error detection and Discord notifications simplify monitoring and troubleshooting.
  </Accordion>
  <Accordion title="Plugin & Addon System">
    Extend functionality through plugins (process-specific) and addons (bot-wide) using Python.
  </Accordion>
  <Accordion title="Compatible with MCDReforged">
    Works alongside MCDReforged without modifying your Minecraft server files.
  </Accordion>
</AccordionGroup>
