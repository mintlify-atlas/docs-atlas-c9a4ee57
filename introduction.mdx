McDis-RCON is a Python application that allows you to relay the console of a Minecraft server to Discord, facilitating remote and efficient management through a Discord bot.

<img src="https://i.imgur.com/lE4GRIV.png" alt="McDis-RCON Panel" />

## What is McDis-RCON?

McDis-RCON bridges the gap between your Minecraft servers and Discord, providing a powerful management interface directly within your Discord server. It streams console output in real-time, allows command execution, and provides comprehensive file management—all without modifying your Minecraft server files.

## Why Choose McDis-RCON?

<AccordionGroup>
  <Accordion title="Non-Invasive Architecture">
    McDis-RCON runs processes using `Popen`, similar to MCDReforged. It doesn't modify your Minecraft server files, making it safe and reversible.
  </Accordion>
  
  <Accordion title="Multiple Launcher Compatibility">
    Works seamlessly with Fabric, Paper, Vanilla, Velocity, Waterfall, Bungeecord, and any Java process.
  </Accordion>
  
  <Accordion title="Multi-Server Management">
    Manage multiple servers and networks simultaneously from a single Discord panel.
  </Accordion>
  
  <Accordion title="Advanced Error Reporting">
    Errors are automatically detected and reported in Discord, simplifying monitoring and troubleshooting.
  </Accordion>
  
  <Accordion title="MCDReforged Compatible">
    Can work alongside MCDReforged. Just ensure `advanced_console` is set to `False` in MCDR configuration.
  </Accordion>
</AccordionGroup>

## Use Cases

### Remote Server Management
Manage your Minecraft servers from anywhere using Discord on desktop or mobile. No need for SSH or direct server access.

### Team Collaboration
Multiple administrators can monitor and control servers through Discord, with all actions visible in the shared console threads.

### Multi-Server Networks
Perfect for managing complex setups with multiple servers and proxy networks (Velocity, Bungeecord, Waterfall).

### Automated Operations
Use plugins to automate tasks, respond to server events, and integrate with other Discord features.

## System Requirements

- Python 3.8 or higher
- Discord bot with administrator permissions
- Linux (recommended) or Windows
- Network access for Discord API

<Note>
  While McDis-RCON can run on Windows, it is primarily designed for Linux systems. Some features may have limited functionality on Windows.
</Note>
